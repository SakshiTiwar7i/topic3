What is an IoT application?

An IoT application or Internet of Things are applications which are used to control the hardware through the use of a simple interface build using web technologies such as python, html, css, js.

-Through this application the user is able to analyse and monitor sensor data.

-These applications can be accessed, used and control the hardware by a multiple number of users.

What are some Examples?

•Smart home security system

•Smart evacuation system

•Health monitoring system

•Hospital or Restaurant Reservation system (Congestion avoiding system)

•Appliance monitor and controller

What are the main Features needed?

•Data Security

•Real time data monitoring

•Scalability

•Integration

•Optimization(Responsiveness)

•Data backup

•User Interface

•Error reports notification

What does smart mean?

Smart building refers to a building which is automated to a certain extent in order to reduce human effort and control appliances using an interface while also covering faults that may occur during the whole process. It also covers resource conservation, security and health of the users.

What are some Smart-building scenarios?

•Security (Intruder detection)

•Congestion avoidance detection

•Air quality monitoring and ventilation

•Turning on and off appliances automatically based on its usage hence saving electricity.

•Contactless appliances

•Smart Evacuation systems

What are the main Features needed?

•Data handling: data security, data backup, data access restriction

•Sensor data precision and accuracy

•Realtime health monitoring

•Fault notifying system

•Health preserving

•Connectivity

•Abstraction

•Automation
